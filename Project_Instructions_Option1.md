# Final Project: Data Visualization Web Application
## Course: Database Design and Implementation (DDI) - Spring 2026

### Overview
For your final project, you will develop a data-driven web application that integrates a backend service, a relational database, and interactive frontend visualizations. The goal is to tell a **data-driven story** through exploration and user interaction.

---

### Tech Stack Requirements
To meet the requirements for **Option 1**, your application must utilize the following technologies:
* **Backend:** Python + Flask
* **Database:** PostgreSQL
* **Frontend:** Interactive visualizations using a supported library. using react and d3.js


---

### Data Requirements
You are required to use **publicly available data**. 

1. I plan to use steam API and SteamDB to get data about video games.

---

### Visualization Requirements
Your application must query your PostgreSQL database and present that data using one of the two supported libraries:

* **D3.js:** A JavaScript library for highly customized, fine-grained interactive visuals. Best for those comfortable with frontend development.

**Format Options:**
* A dashboard of several smaller, coordinated visual widgets.

---

### Project Goal: The "Data Story"
Your project is not a formal scientific analysis. Instead, it should:
* Communicate a specific point of view, experience, or question.
* Prioritize **exploration** over proving a hypothesis.
* Enable users to interact with and understand the dataset's nuances.
